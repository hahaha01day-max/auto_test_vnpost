# Tài liệu 02 - Phân quyền vai trò

Playwright tests sinh từ tài liệu `2. Phân quyền vai trò.docx`.

Phạm vi:
- Mở màn Quản lý vai trò.
- Thêm mới, xem danh sách, tìm kiếm, cập nhật, xóa vai trò.
- Gán quyền chức năng cho vai trò.

Chạy riêng:

```bash
npx playwright test --config tai-lieu-test/02-phan-quyen-vai-tro/playwright.config.js --project=chromium
```
